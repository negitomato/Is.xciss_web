#puc_03 第一商店街

build data 2024/--/-- 座標 x:____ y:____ z:____

本島が文明的になったといえる、通貨「エメラルド」の概念をもたらした。10人も満たない商店ではあるが 農作物や紙の買い取り、ガラスの入手、書士など基本的なところは抑えており、島の経済的発展を促した。

商店を設置するにあたり、原住民監禁所からのびている誘拐用の線路の敷設もpuccaによるものである。

#puc_04 PASMO

build data 2024/--/-- 座標 x:____ y:____ z:____

だれがどう見てもスイカを模した豆腐だが、PASMOである。西瓜と南瓜の栽培所。 第一ゲートのすぐ横という最高の立地により稼働率もかなり高い。

このスイカを模した建物においてはよりスイカブロックに見えるようブロックの見た目・色がかなり厳選されている力作である。

#puc_05 第一ゲート周辺

build data 2024/--/-- 座標 x:____ y:____ z:____

島の東岸に島長xcissが設置した第一ゲートは、ゲートへの歩道と街灯こそ葱戸が追加したものの基本野晒しで 周辺の湧き潰しも甘い状態であった。これを装飾しセーフハウスを用意したのがpuccaで、さらにその後、島の周遊鉄道 である島営横蹴線の始発駅「PASMO前駅」も建築している。

#puc_06 島営横蹴線

build data 2024/--/-- 座標 x:____ y:____ z:____

島の東岸から西岸沖合まで、４駅と信号所１つを備える複線のトロッコ鉄道。 島の周遊鉄道の位置づけではあるが、横蹴線の名にふさわしくほぼ横一直線に島を往貫している。

駅舎はそれぞれ異なったデザインがなされ見た目にも楽しめるほか、高架線であるため島内の様々な建物を見ることができる。

一部の駅をゴーレムが集団で押し入りトロッコが通行できないなど事件もあったが、 駅舎への身長制限を設けたため最近は事なきを得ている。

#puc_07 島営横蹴線／PASMO前駅

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の始発駅。第一ゲートのゲートハウス上に建設されており周囲のジャングル材と砂岩をふんだんに使ったやわらかい色合い。

当駅の駅員がたびたびゾンビ化する事件があり、その都度よるぴによって治療されているが、2025年に入って やっと近辺で明るさが足りなくなっている箇所を発見し、その後は事なきを得ている。

いつの間にか裏手にあるトロッコ搬入用出入口に、東西地下道が接続され、様々な施設へアクセスできるようになっている。

#puc_08 島営横蹴線／島長邸前駅

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の２つ目の駅。とーちょーのおうちのすぐ横に位置するのだが、当初「葱戸邸前」と命名されており開発事務局のクレーム をうけて今の駅名となった。改札をでて目の前が島共有倉庫となっており、駅舎のデザインも砂岩と深層岩を使用したデザインとなっている。

当駅の線路内にゴーレムが５・６体詰めかける事態が何度も発生したため、改札に身長制限のためのフェンスゲートが設置された。 これらの謎の集合については隣のPASMO前駅の駅員ゾンビ化との関連性も含めて調査が行われている。

#puc_09 島営横蹴線／水流エレベータ前駅

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の３つ目の駅。氷ボートからの出入口EVとなるころあき像のすぐ南、植林地下EVの西に位置する。

エンドストーンレンガをふんだんに使用したやや未来的デザインの駅舎には珍しく２階があり、 ゾンビ肉を買い取る駅員と革防具を扱う駅員が勤めている。また、駅舎の外は赤レンガで固められており、 駅の下を通って島の中央と南側を行き来することが可能。

#puc_10 島営横蹴線／第二ネザーゲート前駅

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の最終駅。この先にも線路は伸びているが、駅としてはここまで。島の南西に位置し、 パルプンテが装飾を施した第二ゲートの目の前にある。

海に面しているだけあって、駅員は釣りの達人。非番の日などないので永遠にその腕は披露されないが。

この駅以西は沖合へ進出している。少し離れた位置から島を眺めてみるのはどうだろうか。

#puc_11 島営横蹴線／岩盤ゲート前信号所

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の終着点。ここは駅ではないので駅員はいない。よるぴの開設したネザー岩盤上へのゲートの横にちょっとした 赤レンガ造りの折り返しポイントがあるのみとなっている。

よるぴ敷設のジャッカル倉庫直通路線がこのゲートに侵入しているが、信号所との貨客連絡はなされていない。 ここからネザーへ入る場合は、ラクダに乗って岩盤上施設へ赴くのがいいだろう。

島第一養蜂所からの脱走蜂をつなぎとめるために、よるぴがいくつかの養蜂箱を設置している。

#puc_12 石〇元市長邸跡地

build data 2024/--/-- 座標 x:____ y:____ z:____

島営横蹴線の通路出入口前駅すぐヨコに存在する、謎の史跡。石をあつかう人物の何からしいが、誰もその何かがわからない。

設置されている書物は……気にしないほうがキミの精神は安定するだろうな。

水流とマグマを使用して丸石と焼き石を無限生成できる施設。

#puc_13 島立図書館

build data 2024/--/-- 座標 x:____ y:____ z:____

島北部の中央地、割と一等地にある、２階建ての図書館。オレンジ色のドーム屋根が４分の１ガラス張りになっていてオシャン。

エンチャント施設もあるが、資材も豊富な工房…というよりは飾りつけ重視であり、利用者はすくないか？ ラピスラズリ備蓄が横付けされている、島共有倉庫の地下のエンチャントテーブルのほうが割合利用されている印象。

木製のシャンデアリアがさがり、読書に向いた落ち着いた雰囲気であればだんぜんこちらなのだが。

#puc_14 すいかパン販売所

島東部、第一ゲート出てすぐのところに、店外イートインテーブルを備えたすいかパンの販売所がある。

しかし、ある日その店舗に勤めていた販売員は忽然と姿を消してしまった……店のすぐ裏手には 崩落し陥没したように砂がえぐられていたが、一体何がおこったのか。
