#xci_00 とーちょーのおうち

build data 2024/09/04 座標 x:____ y:____ z:____

世界の北東端でこの砂の島に上陸したxcissがこの島での先ずの住処として建築した木製の豆腐。 基本的には周辺に自生しているジャングルの木によって構成されているが、ところどころにオークや マングローブが混ざり合い本当にあり合わせを用いて一先ずの仮拠点として作成されたことがわかる。

内装も簡素でベッドとLCが１つあるのみ、窓にガラスはなくドアは常に開け放たれている。

玄関横に掲げられた表札は葱戸が作成。のちに入植してきたpuccaがこの建物を島長の邸と 認識しなかったことをうけ、涙しながら掲げたという。

#xci_01 島長の麦畑

build data 2024/09/05 座標 x:____ y:____ z:____

水か回路かをめぐらすようなフチのついた段々畑。そのすべては土でできており、いつみても金の麦穂が風に揺れている。まあ…………ありていに言えば実った状態で放置されている。

おそらくは水流で収穫するのであろう畑であるが、本島開闢から数か月たってもなにも変わらない。変化らしい変化は、いつの折からか土ブロックに草が生えたくらいか。

#xci_02 動物小屋（空中）

build data 2024/09/05 座標 x:____ y:____ z:____

正しくは、原住民の村が生成された段階で、砂の丘の上に精製された構造物の１つ。 これの下にあったはずの砂や砂岩はまるっとぺろっとxcissによって削り取られてしまった。残ったのが、コレ。

のちに、葱戸によって「ネザーゲートからの来訪者がババッと目にみて解るように」Is.XCISSの文字旗が下げられた。

また、誰かしらによっていつの間にか橋がかけられている。

#xci_03 大空洞／大螺旋階段

build data 2024/09/29 座標 x:____ y:____ z:____

本島がxcissの名を冠するべき最大の功績が、この「明らかに島の面積より大きい」大空洞の制圧と 高さにして60ブロック近い高さにそびえる螺旋階段の建築である。

なにかしらの要因で本島の空洞におちても(落下死さえしなければ)命をつなぐことができるのは、ひとえに 危険を顧みず、地道に松明を置いて行ったxcissのおかげである。最初期はそれこそ、地上建築中に地面から ひょっこり島長が出てくる、ログが島長の落下と爆死と戦死で埋まるなど茶飯事であった。

当人の功績と栄誉を讃え、葱戸はこの島をxciss島と称する。みんなもそう呼んでほしい。
