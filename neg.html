#neg_00 葱戸邸

build data 2024/09/06 座標 x:____ y:____ z:____

砂の島を求めた葱戸まとが島長に勧誘され、入植したおりに建設した豆腐小屋。 9x9の深層岩タイルの豆腐が連結してできている。屋上には昆布の自動収穫施設が載っていて屋内に引き込まれており 屋内にて乾燥昆布ブロックに精製されている。これの焼成には乾燥昆布ブロックのそものを燃料としていて、 実際の目的は復帰用の経験値を貯めるものとしている。

入植者が増える前は後述の共用倉庫部分も含めた４つの豆腐で構成されており、 屋上菜園としてトウヒ・南瓜・麦・マングローブが栽培されていたが、それぞれ順次植林ビルに機能が移設され 現在の状態に至る。

また、地下も何度かにわけて拡充されており、現在は地下(y:56)で東西に跨る通路に面するかたちで地下室を 構えているほか、同通路には個人的な装置をいくつも置いていてやりたい放題さが伺える。

#neg_01 島共用倉庫

build data 2024/09/21 座標 x:____ y:____ z:____

もとは葱戸まとの第三・第四豆腐。pucca・よるぴ・パルプンテの3名が入植するにあたり島全体の倉庫の必要性が高まると考え、 まずは第四棟が、そして接続されていた第三も相次いで共用倉庫となった。併設されていたマグマバケツ施設も８槽へ倍化したが その結果、第四棟の屋上であったマングローブ植林が幾度となく炭化することとなりモニュメントに差し替えられている。

その後さらに手狭になった結果、地下にも倉庫がよるぴによって拡充された。地下のチェストの管理運営も葱戸に委譲されている。 拡充されたとはいえ手狭であり、１６色あるガラスや羊毛・テラコッタなどは植林地下３・４階にカラー倉庫として別途備蓄されている。

葱戸がいうには、ジャッカル共用倉庫のようにスカスカにならぬよう常に一定数を備蓄できているかチェックしており、実用性は高いと自負しているとのこと。使え。

#neg_02 植林場（地上）

build data 2024/09/10 座標 x:____ y:____ z:____

#neg_07 植林場地下６階(西)　ネザー系キノコ増産・植林場

build data 2024/10/31 座標 x:____ y:____ z:____

骨粉大量消費して大量の骨粉を作りつつ、ネザー系のキノコを大きくして真紅の幹とゆがんだ幹を得るところ。

地下３階から降りてくる東側のマングローブ植林とは逆に、地下６階で植えて骨粉をかけてから梯子を上って伐採する形。 シュルームライトも結構とれるが幹がメインで幹さえ採取すれば次が生えるので最近は無視されている。

２種類のツタについては需要が低いことから、特段栽培されていない。心の内では作る作る詐欺が行われているが 優先順位が低すぎて永遠に後回しになっている。

植林場地下６階(西)には西に出入口が設けられており、氷ボート路からの地下通路に接続されている。 このため、氷ボート路から島倉庫へはここを通って植林EVホールへ抜けるのが近道である。

#neg_08 植林場水流EVホール

#neg_09 植林場地下７・８階(東)　スライムトラップ

build data 2024/12/01 座標 x:____ y:____ z:____

葱戸が建築インスピレーションを高めるためにブロックの作り方や使い方を纏めた博物館をつくろうと掘り進めるうちに、 植林地下の東側のほとんどを占めるチャンクがスライムチャンクであることが判明し、急遽用意された小型のスライムトラップ。 経験値は取得できない。

南北の壁面に埋め込まれたゴーレムによって階下水流へ誘引し最終的に魂の焚火で仕留める作りだが、水流の敷設が甘く そこに溜まってしまうため粉雪ブロックで追い出すという無理やりぶり。

また、チャンクに食い込むようにアメジストジオードが存在するため、それの自動化のために湧き層が一部切り取られている。

周囲の湧き潰しもそこまで行われていないため効率はさほど出ないものの、ほかの植林地下施設で作業しているうちに 勝手に湧いて勝手に処理され待機の手間がかからない。一応、2025年に入って周辺湧き潰しは行われ始めてはいるが効果のほどは不明。

#neg_23 ゆがんだ監視塔

build data 2024/11/06 座標 x:____ y:____ z:____

ピりジャーが駐屯していそうな形をしたタワーで、ネザーの歪んだキノコの幹をメイン素材として作られている。

頂部の櫓には金ブロックで作った巨大な鐘の中に外敵感知用の鐘と日照センサーが組み込まれており、 センサーを反転させていないため少し曇るくらいでカンカンと煩いぐらい鳴るようになっている。 これがときどき島の北側に湧くピりジャーの襲来にうまくはまり、検知を可能にしている。 なお、原住民の隔離後に設置しているため紐づいておらず、鳴っても影響しない。

#neg_29 東西地下通路／苔→土変換所

build data 2024/12/30 座標 x:____ y:____ z:____

苔ブロックにトウヒを大木で植えてポドゾルに変化させ、それをシルクタッチのないシャベルで壊すことで 土に変換させる施設。砂漠の島であるxciss島にとって、土は貴重であったがこれにより幾許かの余裕が生まれた。

急ぎではないため大樹化までは放置されているが、骨粉と苔ブロックは潤沢にあるため従事すれば それなりの速度で土が作れる。１回の大樹生成で100個ほどの土がとれる。

トウヒについては植林場でも植樹されているため不要ではあるが、なんとなく皮を剥いでためてある。こちらは１回30～40程度。
