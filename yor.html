#yor_01 光るイカスミトラップ

比較的浅いy:40付近から岩盤レベルの高さまでとてつもない広さの大空洞があるが、この一角の水槽にウーパールーパーが飼われており 光るイカが湧いて倒され、自動で回収されて地上の専用施設へ備蓄されている。xciss島の誇る特産品であり、本土には（多分）ない施設である。

#yor_02 ジャッカル共有倉庫直通地下鉄（岩盤経由）

build data 2024/--/-- 座標 x:____ y:____ z:____

よるぴによって敷設されたジャッカル共有倉庫地下とxciss島共有倉庫地下をノンストップで結ぶ特別急行鉄道。 途中、Windmill氏によって解放されたネザー岩盤上を経由する。

途中駅としてxciss島地上駅・ネザー岩盤上駅が存在する。これらで途中下車するには駅構内でトロッコを 乗り捨てる必要があるが、実際にはチャンク読込距離ならちょっと前で乗り捨ててもちゃんとトロッコは回収される。

島の西部から中央にかけて地下のかなり浅いところを地形や建造物にあわせて縫うように走っており、周囲の建造物の地下施設に対しギリギリになっていいることも多い。

#yor_03 ジャッカル共有倉庫直通地下鉄／島倉庫地下駅

build data 2024/--/-- 座標 x:____ y:____ z:____

直通地下鉄の島側の始発駅。島共有倉庫の地下にあり、無限溶岩の脇から降りていくことができる。

地上の島倉庫の延長としてチェストが並んでいるが、これの管理は葱戸に譲渡されている。また、奥には エンチャントテーブルが用意されており、ラピスラズリの備蓄場所も兼ねるほか、島用アレイが飼われており レコード装置もあるため増やすことが可能。アメジストとリードは地上倉庫にあるよ。

いつの間にか東西地下道への通用口が設けられた。

#yor_05 ジャッカル共有倉庫直通地下鉄／ネザー岩盤上行きゲート・二重橋

build data 2024/--/-- 座標 x:____ y:____ z:____

ネザー岩盤上へと通じているゲートは島の西の沖合にあり、そこまで赤レンガによる二重の橋がかかっている。 上段はジャッカル共有倉庫直通地下鉄の線路であり、下段は歩行者用の通路となっている。 歩行者用通路にはソウルサンドが仕込まれており、ソウルスピードのエンチャントがなされている靴があれば 快適に渡れるようになっている。

また、ゲート付近にはpuccaの敷設した島周遊の横蹴線の信号所があり、徒歩で岩盤から訪れた際には 此方を使って島内へ行くことも可能。

島内の建造物ではないが、島の所有物と主張しておく（傲慢）

#yor_07 植林場地下７・８階(東)／大空洞北部　アメジスト自動回収装置

build data 2024/--/-- 座標 x:____ y:____ z:____

上述のスライムトラップに食い込むように存在するアメジストジオードの一部に対し、鍾乳石の成長を感知して アメジストのカケラを採取する装置。鍾乳石の成長はアメジストの芽よりさらに遅い為、 放置していればまあだいたい取れるだろうというおおざっぱな思想のもとに作られている。

事実、ちょうど島共有倉庫の地下に位置することもあってチャンクの読み込まれ頻度が高く、 それなりに採取はできている。シルクタッチを使用しての芽の手動採取も可能なように、 ジオードは東側半分ほどが残されている。

なお、残されている東側は大空洞からみて四半球形に露出しているため、大空洞からもアクセスは可能だが 特に整備されているわけではない。楽に行き来したいならば植林場地下経由をお勧めする。

#yor_08 第二商店街

build data 2024/11/21 座標 x:____ y:____ z:____

xcissがごっそりと掘ったと思われる、空中動物小屋の下のスペースは長らく空白地帯であったが、そこに追加された 新たな商店街。ラーメンが食べれそうなフードコートの様相を呈しているが、主な売り物は様々な毒矢。

中央のフードコート席は筏による４席が設けられており、様々なモブが着席することができるが、 よるぴが招待したスライム（小）は周辺を警らするゴーレムに、ウサギは徘徊する野良猫によって駆逐されてしまっている。

空中動物小屋で食された草ブロックの土パーティクルがポロポロと落ちてくるため、衛生管理が待たれる。

#yor_09 竹プロック自動装置

build data 2024/--/-- 座標 x:____ y:____ z:____

横蹴線の島長邸前駅と共有倉庫直通地下鉄の地上駅の間に位置する、竹の自動回収装置。 大きく育ってから検知伐採し、トロッコで回収したのち自動作業台で竹ブロックに加工されて備蓄される。

竹ブロックへの加工工程は、ある程度竹が溜まらないと実施されないため、未加工の竹を得ることも可能。
